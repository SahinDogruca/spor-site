Evde 3 Günlük Antrenman Programı

1. GÜN: İTİŞ (Göğüs, Omuz, Arka Kol)

  • Dumbbell Floor Press: 4 Set x 15-25 Tekrar Yere sırt üstü uzan, dirseklerin yere değince dur ve göğsünü sıkarak yukarı it.
  • Dumbbell Floor Fly: 4 Set x 15-25 Tekrar Dirseklerinde hafif bir bükülme koru ve göğsünde gerilmeyi hissederek kolları yana aç.
  • Şınav (Push-up): 4 Set x Tükenene Kadar Klasik şınav hareketi, vücudunu düz tut.
  • Diamond Push Ups: 4 Set x 15-25 Tekrar Ellerin birbirine yakın, elmas pozisyonunda şınav.
  • Wide Push Ups: 4 Set x 15-25 Tekrar Ellerin omuzlardan geniş açıkta olduğu şınav varyasyonu.
  • Dumbbell Shoulder Press: 4 Set x 15-25 Tekrar Karın kaslarını sıkarak belini koru ve dambılları başının üzerinde birleştir.
  • Lateral Raises: 3 Set x Tükenene Kadar Dirseklerin hafif bükülü kalsın ve kaldırma işlemini dirseklerinle yönlendir.
  • Bent-Over Reverse Fly: 3 Set x Tükenene Kadar Sırtını düz tutarak öne eğil ve kürek kemiklerini birbirine yaklaştır.
  • Overhead Triceps Extension: 4 Set x 15-25 Tekrar Dirseklerini başına yakın tut ve hareketin alt noktasında tam bir esneme sağla.
  • Triceps Kickback: 4 Set x 15-25 Tekrar Gövdeni sabit tut, sadece ön kolun hareket etsin ve üst noktada arka kolunu sık.
  • Inverse Zottman Curls: 3 Set x Tükenene Kadar Ön kol ve biceps için kombine hareket.

2. GÜN: ÇEKİŞ (Sırt, Biceps, Karın)

  • Bent-Over Dumbbell Row: 4 Set x 15-25 Tekrar Sırtını düz tutarak kalçadan eğil ve dambılları karnına doğru çek.
  • One-Arm Dumbbell Row: 4 Set x Her Kol 15-25 Tekrar Sırtını yere paralel tut ve dambılı kalçana doğru çekerek sırtını sıkıştır.
  • Dumbbell Pullover (Yerde): 3 Set x 15-20 Tekrar Kalçanı aşağıda tut ve dambılı indirirken sırt kaslarındaki gerilmeyi hisset.
  • Reverse Dumbbell Fly: 3 Set x 15-20 Tekrar Kalçanı aşağıda tut ve dambılı indirirken sırt kaslarındaki gerilmeyi hisset.
  • Dumbbell Bicep Curl: 4 Set x 15-25 Tekrar Dirseklerini vücuduna sabitle ve sallanmadan, kontrollü bir kaldırma yap.
  • Hammer Curl: 4 Set x 15-25 Tekrar Avuç içlerin birbirine bakacak şekilde nötr bir tutuş kullan.
  • Concentration Curl: 3 Set x Her Kol Tükenene Kadar Dirseğini bacağının iç kısmına sabitle ve tepe noktasında biceps kasını güçlüce sık.
  • Inverse Zottman Curls: 3 Set x Tükenene Kadar Ön kol ve biceps için kombine hareket.
  • Reverse Grip Curl: 4 Set x 15-25 Tekrar Avuç içlerin aşağı bakacak şekilde dambılları kaldır, ön kolunu da çalıştırır.
  • Lying Leg Raise: 3 Set x 20-30 Tekrar Bacaklarını kontrollü indir, yere değdirmeden tekrar kaldırarak alt karnını çalıştır.
  • Dumbbell Side Bend: 3 Set x Her Yöne 20-25 Tekrar Hareketi sadece yana doğru eğilerek yap, öne veya arkaya bükülme.

3. GÜN: BACAK & KARIN

  • Hip Thrust: 4 Set x 20-30 Tekrar Kalça ve gluteus kaslarını güçlendirir.
  • Goblet Squat: 5 Set x 20-30 Tekrar Göğsünü yukarıda, sırtını düz tut ve olabildiğince derine inmeye çalış.
  • Bulgarian Split Squat: 4 Set x 20-30 Tekrar Tek bacak squat varyasyonu, denge ve güç gerektirir.
  • Dumbbell RDL: 4 Set x 20-30 Tekrar Dizlerini hafif bükülü tut, sırtını düz tutarak kalçanı geriye it.
  • Dumbbell Lunge: 4 Set x Her Bacak 15-20 Tekrar Adım attığında her iki dizinin de 90 derecelik açı oluşturmasını hedefle.
  • Plie Squat: 4 Set x 20-30 Tekrar Ayakların geniş ve dışa dönük olsun, iç bacaklarındaki gerilmeyi hisset.
  • Dumbbell Glute Bridge: 4 Set x 25-35 Tekrar Dambılı kalçanın üzerine yerleştir, tepe noktasında kalçanı 2 saniye sık.
  • Reverse Lunge to Lateral Lunge: 4 Set x 10-15 Tekrar Kombine lunge hareketi, tüm bacak kaslarını çalıştırır.
  • Russian Twist: 3 Set x Her Yöne 20-25 Tekrar Dönüşleri omuzlarından değil, gövdenden yaparak yan karın kaslarını hedefle.
  • Dumbbell Crunch: 3 Set x 20-30 Tekrar Belinin yerden kalkmamasına özen göster, hareketi yavaş ve kontrollü yap.
  • Plank: 3 Set x Maksimum Süre Kalçanı sıkarak vücudunu baştan topuklara kadar düz bir çizgi halinde tut.

Önemli Notlar

  • Isınma & Soğuma: Her antrenmandan önce 5-10 dakika dinamik ısınma, sonrasında ise 5 dakika statik esneme yapmayı unutma.
  • Form Odaklı Ol: Videolardaki formu dikkatle izle. Ağırlığı artırmaktan çok, hareketi doğru yapmak önemlidir.
  • Zorluk Seviyesi: Hareketler kolay gelmeye başladığında tekrar sayılarını artırabilir, setler arası dinlenmeyi kısaltabilir veya daha zorlayıcı varyasyonlara geçebilirsin.
  • Dinlenme: Setler arasında 60-90 saniye dinlen.
